# Troubleshooting

This page records sanitized setup lessons. Do not add local paths, project names, usernames, private endpoints, raw logs, or private operational details.

## Claude MCP Add Reports Invalid Environment Variable Format

Use the command generated by this repository after the CLI argument-order fix:

```sh
claude mcp add -s local -e SEARXNG_URL=http://127.0.0.1:8080 -t stdio searxng -- npx -y mcp-searxng
```

The environment option is variadic in Claude Code. Keep another option such as `-t stdio` between `-e KEY=value` and the server name so the server name is not parsed as another environment value.

## JSON Works But Search Returns Zero Results

This means the SearXNG JSON API is reachable, but the selected search engines did not return usable results.

Use a profile that works in your region:

```sh
npm run setup:searxng -- --profile bing-only --apply --start
make verify-search URL=http://127.0.0.1:8080
```

Do not document regional service workarounds. The correct fix is to select lawful, reachable search engines for the user's region.

## First Search Is Slow

Use a longer timeout and retries:

```sh
make verify-search URL=http://127.0.0.1:8080 TIMEOUT_MS=30000 RETRIES=2
```

## Container Is Still Starting

Check service status:

```sh
docker compose ps
docker compose logs searxng
```

Wait for the service to finish starting, then retry `make verify-search`.

## Docker Compose Uses A Compatibility Provider

Some systems provide Docker-compatible commands through another container runtime. This can still work, but behavior may differ from Docker Desktop.

Check:

```sh
docker --version
docker compose version
docker-compose version
podman-compose version
```

Use the command that is valid for the user's local container runtime. Keep troubleshooting notes generic and do not include machine-specific paths.

## MCP Connected But Search Still Needs Validation

`claude mcp list` and `claude mcp get searxng` only prove that Claude Code can connect to the MCP server.

Also run:

```sh
make verify-search URL=http://127.0.0.1:8080
```

Then ask Claude Code to perform a public-safe search and cite opened public URLs.
